# GitHub Pages Starter

A simple starter template for GitHub Pages.

## Run Locally

Simply open:

```html
index.html
```

in your browser.

## Deploy

1. Create a repository named:

```
YOUR_USERNAME.github.io
```

2. Upload all files.

3. Commit and push.

4. Visit:

```
https://YOUR_USERNAME.github.io
```

Your site should be live within a few minutes.
